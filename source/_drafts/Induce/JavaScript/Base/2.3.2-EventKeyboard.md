Web 浏览器中可能发生的事件有很多类型。如前所述，不同的事件类型具有不同的信息，而 DOM3 级事件规定了以下几类事件。

名称 | 描述
-------|------
keydown | 当用户按下键盘上的任意键时触发。按住不放，会重复触发。
keypress | 当用户按下键盘上的字符键时触发。按住不放，会重复触发。
keyup | 当用户释放键盘上的键时触发。
